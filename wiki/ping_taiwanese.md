# [台灣] C Shell (csh) ping 用法: 測試網路連通性

## Overview
`ping` 命令用於測試網路連通性，透過向指定的主機發送 ICMP 回音請求來檢查其是否可達。

## Usage
基本語法如下：
```
ping [選項] [主機名稱或 IP 位址]
```

## Common Options
- `-c <次數>`: 指定發送的 ICMP 回音請求數量。
- `-i <間隔>`: 設定發送請求的間隔時間（以秒為單位）。
- `-s <大小>`: 設定發送的數據包大小。

## Common Examples
以下是一些常見的使用範例：

1. **測試與 Google 的連通性**
   ```csh
   ping google.com
   ```

2. **發送 5 次請求**
   ```csh
   ping -c 5 google.com
   ```

3. **每 2 秒發送一次請求**
   ```csh
   ping -i 2 google.com
   ```

4. **發送指定大小的數據包**
   ```csh
   ping -s 128 google.com
   ```

## Tips
- 使用 `-c` 選項可以避免無限迴圈，方便測試。
- 在測試時，注意網路的穩定性，避免在高峰時段進行測試。
- 對於不同的主機，結果可能會有所不同，建議多測試幾個不同的地址以獲得更全面的了解。
