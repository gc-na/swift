# [操作系统] C Shell (csh) 默认命令: [显示当前用户的主目录]

## 概述
`pwd` 命令用于显示当前工作目录的完整路径。在使用命令行时，了解你当前所在的目录是非常重要的，尤其是在处理文件和目录时。

## 用法
基本语法如下：
```
pwd [options]
```

## 常用选项
- `-L`：显示逻辑路径，即符号链接的路径。
- `-P`：显示物理路径，即实际的路径，不包含符号链接。

## 常见示例
1. 显示当前工作目录：
   ```csh
   pwd
   ```

2. 显示逻辑路径：
   ```csh
   pwd -L
   ```

3. 显示物理路径：
   ```csh
   pwd -P
   ```

## 小贴士
- 在脚本中使用 `pwd` 可以帮助你确认当前目录，避免路径错误。
- 结合其他命令使用，例如 `cd`，可以更有效地管理文件和目录。
